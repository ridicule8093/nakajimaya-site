# 画像格納フォルダ

このフォルダに中島屋サイト用の画像を入れます。

## 入れる画像ファイル

以下のファイル名でアップロードしてください。

- `hero-pallets.jpg`
- `company-yard.jpg`
- `product-pallets.jpg`
- `product-lumber.jpg`
- `product-logs.jpg`
- `product-plywood.jpg`
- `product-boards.jpg`
- `warehouse-wide.jpg`
- `warehouse-pallets.jpg`

## 注意

ファイル名が変わると、HTML/CSS側で画像が表示されません。

トリミング済み画像セットのZIPを解凍し、中にある `assets/img/` の画像ファイルだけをこのフォルダへアップロードしてください。
