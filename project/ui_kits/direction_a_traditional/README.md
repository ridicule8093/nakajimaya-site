# 方向A — 堅実な木材会社・製材業寄り

## 一言で

「材木店」「製材業」の系譜にある老舗のたたずまい。明朝で見出しを組み、罫線で構造を見せる。色は紙色＋墨＋焦茶で、緑は控えめ。落款のような小さな署名（写真右下）でドメイン感を出す。

## 採用したい時

- 「初取引の相手が見ても "ちゃんとした会社らしさ" を最優先したい」
- 「同業他社と並んだ時に、地に足のついた印象で選ばれたい」
- 木材を扱う事業に対する敬意を、書体と罫線で表現したい

## 主な意匠

- **見出し**: Noto Serif JP 700。日本語の明朝特有の "縦のリズム" が、地場・実在感を作る
- **ロゴ**: ワードマーク＋下部の小さなラテン銘（NAKAJIMAYA · est. AICHI）。シンボルマークは作らない
- **角丸**: 2-4px の極小。木材を切り出した感
- **写真**: 中サイズ（ヒーロー 420px・商品サムネ 140px）。文字組と等価値
- **構造**: 商品一覧は「罫線テーブル」状で並べる（仕様欄を右に常設）
- **トップ帯**: 黒地に英文ラベル — パンくず的ナビゲーション（古い商社サイト風）

## 写真の使い方

- ヒーロー = `warehouse-pallets.jpg`（屋内のパレット山）
- 商品サムネ = 各 product-*.jpg をそのまま
- 「その他木材」は写真を当てず、罫線枠＋「その他」テキストの相談枠扱い（依頼明記）

## 避けたこと

- 大きすぎる角丸
- 強い影
- フルブリードの写真コラージュ（カフェ・住宅会社寄りになる）
- 緑のグラデーション
- 絵文字・装飾アイコン

## 実装メモ

- すべて静的HTML/CSS。JavaScript なし
- `style.css` の `:root` 変数は `colors_and_type.css` と一致
- 既存リポジトリの `assets/css/style.css` を置換せず、ここを参照して該当箇所を順次置き換える形を推奨
