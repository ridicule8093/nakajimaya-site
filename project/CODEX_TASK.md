# Codex 実装指示

このリポジトリの `SITE_SPEC.md` と `CONTENT.md` を読み、株式会社中島屋の静的コーポレートサイトを作成・修正してください。

## 実装目的

株式会社中島屋の会社案内サイトを、静的HTML/CSSで作成する。

大がかりなWebアプリではなく、BtoB中小企業の会社案内として、事業内容、取扱商品、会社概要、問い合わせ導線が分かるサイトにする。

## 作成するページ

- `index.html`：トップページ
- `products.html`：取扱商品ページ
- `company.html`：会社概要・お問い合わせページ

## 技術条件

- HTML/CSSで作成する
- フレームワークは使わない
- ビルドツールは使わない
- 静的ファイルとしてそのまま動作する
- Cloudflare PagesまたはGitHub Pagesで公開できる構成にする
- スマホ、タブレット、PCに対応するレスポンシブデザインにする
- SEOの基本としてtitle、meta description、h1/h2構造を適切に設定する
- 画像は仮置きでよいが、後から差し替えやすい構造にする

## デザイン条件

中小企業のBtoBサイトとして自然な見え方にする。

重視する点：

- 落ち着いた信頼感
- 木材、パレット、物流、製造業の実務感
- 読みやすさ
- 問い合わせ導線の分かりやすさ
- スマホでの見やすさ

避ける点：

- スタートアップ風
- SaaS風
- 派手なアニメーション
- 抽象的なビジュアル
- 過度におしゃれなデザイン
- 汎用的で実態のないコピー

## 文言条件

`CONTENT.md` の文章をベースにする。

事実を追加で推測しない。

電話番号、メールアドレス、問い合わせフォームは未設定のため、仮表示またはコメントで分かるようにする。

## 画像の扱い

画像は初期状態ではCSSのプレースホルダーでよい。

外部画像を勝手に使用しない。

後から以下の写真に差し替えやすくする。

- 木製パレット
- パレットの積み上げ
- 木材資材
- 工場、倉庫、作業場の雰囲気
- トラック、物流、納品前の資材
- 会社外観

## 禁止事項

- 実態不明の実績を追加しない
- 取引先名を勝手に出さない
- 認証、保証、許認可、品質規格を勝手に記載しない
- 「業界トップ」「最高品質」「全国対応」など根拠のない表現を使わない
- AI生成っぽい抽象コピーを増やさない
