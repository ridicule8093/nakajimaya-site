# 公開前チェックリスト

中島屋サイトを公開前に確認するためのチェックリストです。
コード上の整合確認ではなく、実際にブラウザで表示して確認する項目です。

## 1. 画像確認

確認ページ：`dev-image-check.html`

- [ ] `warehouse-pallets.jpg` がトップ・お問い合わせ用の写真として自然に見える
- [ ] `warehouse-wide.jpg` が会社概要用の写真として自然に見える
- [ ] `product-pallets.jpg` が木製パレットの写真として合っている
- [ ] `product-lumber.jpg` が角材・板材の写真として合っている
- [ ] `product-plywood.jpg` が合板・ベニヤの写真として合っている
- [ ] `product-logs.jpg` が丸太杭・丸太の写真として合っている
- [ ] 写真の切れ方が不自然ではない
- [ ] 商品名と写真がずれていない

## 2. PC表示確認

確認対象：

- `index.html`
- `products.html`
- `company.html`
- `contact.html`

確認項目：

- [ ] トップページのファーストビューで写真と文章のバランスが悪くない
- [ ] トップページの主な取扱商品が4枚カード＋その他相談ボックスで自然に見える
- [ ] 取扱商品ページの各商品欄が見やすい
- [ ] 会社概要ページが薄すぎず、不自然に寂しく見えない
- [ ] お問い合わせページで、電話・メール・所在地が分かりやすい
- [ ] フッターのリンク・文言に違和感がない

- [ ] 上部の黒いバーが窮屈すぎない

- [ ] トップの写真が大きすぎず、小さすぎない
